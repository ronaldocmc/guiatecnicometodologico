# Técnica

## Definição

Uma técnica é a forma operacional de aplicar um método.

## O que especificar

Defina problema tratado, parâmetros, condições, impacto esperado e limitações.

## Como implementar ou demonstrar

Demonstre comparação antes/depois, com dados de exemplo e métricas de impacto.

## Como validar

Valide pelo efeito produzido, efeitos colaterais, reprodutibilidade e adequação ao domínio.

## Artefatos esperados

- descrição formal;
- exemplos aplicados;
- metadados mínimos;
- critérios de qualidade;
- evidências de validação;
- registro de limitações.

## Exemplos no guia

SMOTE, anonimização, SHAP, LIME, PSI, prompt few-shot, TF-IDF, embeddings, normalização, imputação.
