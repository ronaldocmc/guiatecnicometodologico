# Framework

## Definição

Um framework é uma estrutura organizada e reutilizável para tratar uma classe de problemas.

## O que especificar

Defina classe de problemas, princípios, módulos, processos, artefatos, critérios de aderência e exemplos de instanciação.

## Como implementar ou demonstrar

Implemente por documentação, templates, checklist, exemplos, repositório e estudo de caso.

## Como validar

Valide por aderência, completude, aplicabilidade, reuso, utilidade e rastreabilidade.

## Artefatos esperados

- descrição formal;
- exemplos aplicados;
- metadados mínimos;
- critérios de qualidade;
- evidências de validação;
- registro de limitações.

## Exemplos no guia

EdmLens: engenharia de dados educacionais. SemED-LLM: diagnóstico semântico. GeoReXAI: modelagem geoespacial explicável.
