# Roteiro rápido para o aluno

## 1. Defina o tipo de proposta

Use a página [Tipos de proposta](../tipos/index.md).

## 2. Escreva a lacuna

Use a fórmula:

> Embora existam abordagens para X, ainda há limitação em Y, especialmente quando Z.

## 3. Defina a contribuição

> Esta pesquisa propõe um(a) [tipo de proposta] para [classe de problema], composto(a) por [elementos], com validação por [método de avaliação].

## 4. Defina a implementação mínima

A implementação deve demonstrar o fluxo principal, mesmo que em escala reduzida.

## 5. Defina a validação

A validação deve conter:

- questão de avaliação;
- critério;
- métrica;
- instrumento;
- evidência;
- interpretação.
