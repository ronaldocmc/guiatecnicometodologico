# Métricas e validação

Cada critério precisa ser convertido em métrica, indicador, instrumento e evidência.

| Critério | Métrica | Instrumento | Evidência |
|---|---|---|---|
| Completude | itens atendidos / previstos | checklist | matriz preenchida |
| Rastreabilidade | relações rastreáveis / esperadas | grafo de linhagem | trilha fonte-saída |
| Usabilidade | SUS ou Likert | questionário | respostas |
| Desempenho | F1, AUC, RMSE, tempo | experimento | resultados |
| Reprodutibilidade | execuções reproduzidas / testadas | reexecução | logs |
