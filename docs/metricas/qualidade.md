# Métricas de qualidade

- Completude: campos preenchidos / campos esperados.
- Consistência: registros consistentes / registros totais.
- Validade: registros válidos / registros totais.
- Unicidade: registros únicos / registros totais.
- Integridade: relações válidas / relações esperadas.
- Atualidade: registros atualizados / registros totais.
