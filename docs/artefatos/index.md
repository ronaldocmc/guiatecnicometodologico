# Artefatos e documentação

Toda proposta deve definir quais artefatos produz, consome e valida.

| Categoria | Exemplos |
|---|---|
| Entrada | fontes, datasets, textos, mapas, documentos |
| Processo | logs, scripts, notebooks, prompts, parâmetros |
| Intermediários | datasets tratados, features, classificações, matrizes |
| Saída | relatórios, dashboards, modelos, mapas, recomendações |
| Governança | matriz de acesso, finalidade, políticas, aprovações |
| Reprodutibilidade | manifesto, ambiente, versões, pacotes |
