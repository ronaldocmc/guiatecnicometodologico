# Reprodutibilidade

Uma proposta reprodutível registra dados, código, ambiente, parâmetros, seeds, versões, modelos, prompts e logs.

## Pacote de reprodutibilidade

- README.
- Dados de exemplo ou instruções de acesso.
- Scripts.
- Ambiente (`requirements.txt`, `environment.yml` ou Dockerfile).
- Configurações.
- Seeds.
- Logs.
- Resultados esperados.
