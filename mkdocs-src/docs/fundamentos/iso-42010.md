# ISO/IEC/IEEE 42010 e descrição arquitetural

A descrição arquitetural deve identificar stakeholders, concerns, viewpoints, views e modelos arquiteturais.

| Conceito | Uso no guia |
|---|---|
| Stakeholder | parte interessada |
| Concern | preocupação relevante |
| Viewpoint | convenção para construir visão |
| View | representação concreta |
