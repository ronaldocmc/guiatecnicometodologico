# Método

## Definição

Um método é um procedimento sistemático para realizar uma tarefa específica.

## O que especificar

Defina tarefa, entrada, saída, condições de uso, parâmetros, pseudocódigo, baseline e métricas.

## Como implementar ou demonstrar

Implemente como função, script, notebook, algoritmo ou serviço simples que execute a tarefa.

## Como validar

Valide por efetividade, eficiência, robustez, estabilidade, comparação com baseline, análise de erro e reprodutibilidade.

## Artefatos esperados

- descrição formal;
- exemplos aplicados;
- metadados mínimos;
- critérios de qualidade;
- evidências de validação;
- registro de limitações.

## Exemplos no guia

FairED-Pipeline: método de mitigação de vieses. SemED-LLM: método de classificação semântica. EdmLens: método de detecção de drift.
