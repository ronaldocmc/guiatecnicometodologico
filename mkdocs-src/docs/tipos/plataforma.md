# Plataforma

## Definição

Uma plataforma é um ambiente computacional integrado que oferece funcionalidades a usuários ou sistemas.

## O que especificar

Defina perfis de usuário, funcionalidades, dados, interface, segurança, logs, tarefas e requisitos.

## Como implementar ou demonstrar

Implemente interface, backend/API, banco ou arquivos estruturados, autenticação quando necessário e logs de uso.

## Como validar

Valide por funcionalidade, efetividade, eficiência, navegabilidade, usabilidade, segurança, utilidade e desempenho.

## Artefatos esperados

- descrição formal;
- exemplos aplicados;
- metadados mínimos;
- critérios de qualidade;
- evidências de validação;
- registro de limitações.

## Exemplos no guia

UnespDataLens como wiki/plataforma de conhecimento; MILA-eDU como plataforma analítica; GeoReXAI como plataforma geoespacial.
