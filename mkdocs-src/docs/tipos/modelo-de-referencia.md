# Modelo de referência

## Definição

Um modelo de referência é uma estrutura abstrata, reutilizável e generalizável que orienta diferentes instanciações de uma classe de soluções.

## O que especificar

Defina princípios, componentes, artefatos, metadados, contratos, relações, critérios e dimensões transversais.

## Como implementar ou demonstrar

Implemente com documento formal, templates, checklist de aderência e pelo menos uma instanciação demonstrativa.

## Como validar

Valide por cobertura de lacunas, cobertura de elementos, aplicabilidade, generalidade, redundância e avaliação por especialistas.

## Artefatos esperados

- descrição formal;
- exemplos aplicados;
- metadados mínimos;
- critérios de qualidade;
- evidências de validação;
- registro de limitações.

## Exemplos no guia

EdmLens: modelo de referência para pipelines educacionais. MILA-eDU: modelo de referência para apoio à decisão. UnespDataLens: modelo de referência para organização de conhecimento.
