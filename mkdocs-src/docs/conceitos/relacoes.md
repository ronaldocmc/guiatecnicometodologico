# Relações entre conceitos

## Arquitetura, pipeline e framework

- A arquitetura mostra **como a solução está organizada**.
- O pipeline mostra **como o fluxo executa**.
- O framework integra **estrutura, processos, artefatos, critérios e reuso**.

## Método, técnica e metodologia

- Método: procedimento sistemático para tarefa específica.
- Técnica: forma operacional de aplicar um método.
- Metodologia: organização de etapas, métodos, critérios e instrumentos.

## Modelo e modelo de referência

- Modelo representa um fenômeno, processo ou estrutura.
- Modelo de referência orienta múltiplas instanciações de uma classe de soluções.

## Plataforma e artefato

- Plataforma é um ambiente computacional com funcionalidades.
- Artefato é qualquer produto gerado pela pesquisa.
