# Problemas tratados

## Problemas de dados

- Dados ausentes.
- Dados duplicados.
- Dados inconsistentes.
- Dados obsoletos.
- Dados sensíveis.
- Dados sem responsável.
- Dados sem documentação.
- Dados com baixa representatividade.

## Problemas de integração

- Silos de dados.
- Chaves incompatíveis.
- Granularidade incompatível.
- Conflitos semânticos.
- Categorias divergentes.
- Schema drift.
- Data contracts ausentes.

## Problemas analíticos

- Data overload.
- Data debt.
- Label quality.
- Data uncertainty.
- Data leakage.
- Desbalanceamento de classes.
- Viés algorítmico.
- Baixa reprodutibilidade.

## Problemas operacionais

- Pipeline manual.
- Falha silenciosa.
- Ausência de logs.
- Ausência de monitoramento.
- Ausência de versionamento.
- Baixa observabilidade.
- Dificuldade de auditoria.
