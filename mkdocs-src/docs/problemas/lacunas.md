# Lacunas

| Tipo | Exemplo |
|---|---|
| Conceitual | falta de distinção entre método, técnica, metodologia e framework |
| Metodológica | ausência de roteiro formal para conduzir análise |
| Técnica | baixa automação de metadados e logs |
| Arquitetural | pouca integração entre dados, governança e modelagem |
| Avaliativa | ausência de métricas para rastreabilidade e reprodutibilidade |
| Semântica | conceitos e variáveis pouco documentados |
| Ética/governança | uso de dados sensíveis sem finalidade explícita |
| Reprodutibilidade | dificuldade de reconstruir experimentos e pipelines |
